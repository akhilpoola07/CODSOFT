# Task 2 - Calculator Application

This project is a simple Calculator application created for the CODSOFT Python Programming Internship.

## Features
- Perform addition, subtraction, multiplication, and division
- CLI version for terminal use
- GUI version using Tkinter
- Handles invalid input and division by zero

## Tools Used
- Python
- Terminal / Command Prompt
- Tkinter

## How to Run CLI Version
1. Open the project folder in terminal.
2. Run:

```bash
python calculator_cli.py
```

## How to Run GUI Version
```bash
python calculator_gui.py
```

## Project Files
- `calculator_cli.py` - command-line calculator
- `calculator_gui.py` - GUI calculator using Tkinter
- `README.md` - Task 2 documentation
