# Task 1 - To-Do List Application

This project is a Python To-Do List application created for the CODSOFT Python Programming Internship.

## Features
- Add new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- Save tasks automatically in `tasks.txt`

## Tools Used
- Python
- Terminal / Command Prompt
- Tkinter (optional GUI version)

## How to Run CLI Version
1. Open the project folder in terminal.
2. Run:

```bash
python todo_cli.py
```

## How to Run GUI Version
```bash
python todo_gui.py
```

## Project Files
- `todo_cli.py` - command-line version
- `todo_gui.py` - GUI version
- `tasks.txt` - stores tasks automatically after use

## Demo Ideas
- Add 2 or 3 tasks
- View the task list
- Update one task
- Delete one task
- Exit and reopen to show saved tasks
